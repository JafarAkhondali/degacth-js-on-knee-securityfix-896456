Поддержка различных языков мира (юникод)

  • для поддержки юникод используется флаг "u"
  • для поиска символов юникод используется синтаксис \p{SomeName}
  • Буквы L:
    • в нижнем регистре Ll,
    • модификаторы Lm,
    • заглавные буквы Lt,
    • в верхнем регистре Lu,
    • прочие Lo.
  • Числа N:
    • десятичная цифра Nd,
    • цифры обозначаемые буквами (римские) Nl,
    • прочие No.
  • Другие категории символов

Стрелочные функции

  • function add(a, b) { return a + b } == const add = (a, b) => a + b
  • function onChange() { alert('it changed') } == const onChange = () => { alert('it changed') }

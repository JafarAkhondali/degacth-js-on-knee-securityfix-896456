Выражения (Expression) и Function Declaration

  • Выражения:
    • const name = 'Alex Monad'
    • const hello = function (name) {return 'Hello ' + name + '!'}
    • const helloMessage = hello(name)
    • let mod = someNum < 0 ? someNum * -1 : someNum
    • const something = ...
  • Не выражения
    • function someName() {}
    • if (true) {} else {}
    • for (;;) {}
    • while (false) {}
    • const result = ???
  • Hoisting - подъём имени переменной или функции
